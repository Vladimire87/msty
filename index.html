Quick Guide: How to fill the form

  • S — Current shares: total number of MSTY shares you already own. Fractional shares are OK.
  • C — Total cost basis ($): how much you’ve spent so far on this position. If unknown, leave it empty and use Avg.
  • Avg — Current average price ($): only used when C is empty; the calculator uses C = S × Avg.
  • P — Buy price ($): auto‑filled from the live quote; adjust if you plan a different price.
  • T — Target average ($): the new average price you want to reach after buying more shares.
  • Strict < T: when checked, result guarantees the new average is strictly below T; when unchecked, ≤ T is allowed.

Tip: If the result looks borderline due to rounding, the tool will suggest adding 1 extra share for safety.

Formula (when P < T): for strict “< T” use ⌊(C − T·S)/(T − P)⌋ + 1; for “≤ T” use ⌈(C − T·S)/(T − P)⌉. If C is omitted, we use C = S·Avg. Fees/taxes ignored.
